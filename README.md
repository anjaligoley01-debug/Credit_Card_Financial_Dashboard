# Credit_Card_Financial_Dashboard
Power bi Dashboard
Project Overview - This project is a POWER BI dashboard designed to analyze credit card transactions and customer data. It provides meaningful insights into revenue,  spending patterns, customer behavior, and overall financial performance. 
Objective - The main objective of this project is to build an interactive dashboard that: Track credit card transaction, Analyzes customer demographics and Monitor revenue and financial trends.
Features - Revenue analysis, customer segmentation and interactive dashboard.
Tools Used - Powerbi and Microsoft Excel / csv.
Dataset - The dataset used in this project includes: Credit card transaction data and Customer data.
Conclusion - This project helped me understand Power bi dashboard and data visualization.
