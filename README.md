# Credit_Card_Financial_Dashboard

## Project Overview
This project is a Power BI dashboard designed to analyze credit card transactions and customer data. It provides meaningful insights into revenue, spending patterns, customer behavior, and overall financial performance.

The dashboard helps in understanding key metrics and supports data-driven decision-making.

## Objective
The main objective of this project is to build an interactive dashboard that:
- Tracks credit card transactions
- Analyzes customer demographics
- Monitors revenue and financial trends
- Provides actionable business insights

## Features
- Interactive and user-friendly dashboard
- Revenue and transaction analysis
- Customer segmentation (gender, income, etc.)
- Data visualization using charts and KPIs

##  Tools & Technologies Used
- Power BI  
- Microsoft Excel / CSV  
- DAX (Data Analysis Expressions)

##  Dashboard Insights
- Analysis of total revenue and transaction amount  
- Customer contribution (male vs female)  
- Identification of top-performing card types
